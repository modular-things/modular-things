##### `async loop(fn, minInterval = 0)`

Run a function `fn` at least every `minInterval` milliseconds.

##### `async sleep(ms)`

Delay for `ms` milliseconds.

##### `render(node)`

Render the DOM node `node` to the View tab.

##### `viewEl`

A reference to the root node of the View tab.
